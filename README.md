# 好用的 Chrome 插件

### Proxy SwitchyOmega

代理工具，很方便定义规则。

一个规则列表：<https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt>

### Vimium

用 vim 的快捷键进行网页浏览。

### Wappalyzer

分析网站技术栈

### Postman

API 请求测试

### Take Webpage Screenshots Entirely - FireShot

网页截图

### Momentum

高大上主题

### 有道云笔记网页剪报

将网页保存到有道云笔记。印象笔记也有类似插件。
